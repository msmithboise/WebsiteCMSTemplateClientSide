New icons feature updated!

We changed the way our icons feature works. Click to add an icon and use the dropdown to select the desired social media. Then use the edit settings menu to adjust the font size.


Developer Notes:

The original Icon feature did not have a way of changing the color at first, we had to find a new approach to make this possible so that users with dark backgrounds could have more visible icons.
